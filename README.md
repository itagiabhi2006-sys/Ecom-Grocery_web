# KS-Backend - E-commerce Backend Application

## Tech Stack
- Spring Boot 3.5.5
- MySQL Database
- JWT Authentication
- OAuth2 (Google & GitHub)
- Razorpay Payment Gateway
- Cloudinary for Image Storage
- ML Service Integration

## Prerequisites
- Java 24 or higher
- MySQL Server
- Maven
